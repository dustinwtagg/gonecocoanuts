Need to make a fast booking on your Google Calendar without logging into WooCommerce? We can help!

== Description ==
Do not be ashamed that you have to use your WooCommerce login to create a Booking when using your Google Calendar app is so much easier on the fly. For everyone out there who does not want to spend all day in the WordPress back end - we have fixed the problem for you! <strong>Absolutely free!</strong>

== Installation ==
1. Upload `google-calendar-woocommerce-bookings-two-way-sync` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Visit Ninja Bookings and enter your information exactly like you did with WooCommerce Bookings

== Frequently Asked Questions ==
= Q: Do I have to have WooCommerce Bookings Installed? =
A: Yes, you can buy it from the wonderful team at WooCommerce.
